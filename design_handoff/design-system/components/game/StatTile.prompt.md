**StatTile** — a single stat in a profile or standings grid.

```jsx
<StatTile label="Tournament wins" value="14" icon={<Icon name="trophy" />} delta="3" />
<StatTile label="Avg KDA" value="4.8" />
```
Big mono tabular value. Optional `icon` and `delta` (`deltaDir="up"|"down"`). Lay several out in a CSS grid for the Arena Stats block.
