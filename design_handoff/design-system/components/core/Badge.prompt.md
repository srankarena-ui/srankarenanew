**Badge** — compact status pill (mono, uppercase) for tournament/match state.

```jsx
<Badge variant="accent">Open</Badge>
<Badge variant="danger" pulse>Live</Badge>
<Badge variant="success" dot>Ended</Badge>
<Badge variant="outline" size="sm">Draft</Badge>
```
Variants: default, accent, success, warning, danger, info, outline. `pulse` for live indicators, `dot` for a static status dot.
