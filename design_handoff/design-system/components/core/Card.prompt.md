**Card** — the universal surface container for any grouped content.

```jsx
<Card hover>…tournament…</Card>
<Card as="a" href="#" interactive padding="lg">…</Card>
<Card inset padding="sm">nested panel</Card>
```
`padding` none/sm/md/lg · `hover` lifts content cards · `interactive` adds accent border + glow for clickable cards · `inset` for recessed nested panels.
